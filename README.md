## 📌 About

This repository contains the **Admin Dashboard** for an AI-powered e-commerce platform (men's clothing store). The dashboard is built with **Vue.js 3** and provides full control over products, orders, and customers.

> The full system also includes a **Flutter mobile app** and an **AI recommendation engine**, each maintained in separate repositories.

---

## ✨ Dashboard Features

- 🔐 Admin login & authentication
- 📦 Full product management (add, edit, delete, image upload)
- 🛒 Order tracking and status updates
- 👥 Customer management
- 📊 Sales reports and analytics
- 🧾 Invoice generation and export (PDF)
- 📁 Spreadsheet export (Excel)
- 🗺️ Map integration (Leaflet)

---

## 🛠️ Tech Stack

| | Technology |
|--|-----------|
| Framework | Vue.js 3 |
| Build Tool | Vite |

| Routing | Vue Router |
| UI | Bootstrap 5 |
| HTTP Client | Axios |
| PDF Export | jsPDF + jsPDF-AutoTable |
| Excel Export | SheetJS (xlsx) |
| Maps | Leaflet.js |
| Deployment | GitHub Pages |

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/E-Commerce-Platform-Graduation-Project/E-Commerce-Frontend.git

# 2. Install dependencies
npm install

# 3. Run in development mode
npm run dev
```

> **Note:** The backend API runs over HTTP. To use the dashboard locally, make sure you have access to the backend server.
